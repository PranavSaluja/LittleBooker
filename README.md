# 🎪 Little Booker

**A Next.js-powered ticket booking platform designed exclusively for children's entertainment**

## 🎯 Vision

Little Booker aims to become the go-to platform for booking children's entertainment, competing directly with BookMyShow in the kids' content vertical. We're building a secure, intuitive, and fun booking experience that parents trust and children love.

## 🌟 What We're Building

A comprehensive ticket booking platform featuring:

- 🎭 **Kids-First Content**: Puppet shows, magic performances, workshops, and movies
- 💺 **Smart Seat Selection**: Interactive seat maps with real-time availability
- 🔒 **Secure Booking Flow**: From seat selection to payment confirmation
- 📱 **Mobile-Optimized**: Kid-friendly design that works perfectly on all devices
- 💳 **Safe Payments**: Razorpay integration for secure transactions
- 🎫 **Digital Receipts**: Instant booking confirmations and management

## 🛠️ Tech Stack

**Frontend**
- Next.js 14+ with App Router
- React 18+ with TypeScript
- Tailwind CSS for styling
- Responsive, mobile-first design

**Backend & Database**
- Firebase Authentication
- Firestore NoSQL database
- Next.js API routes
- Real-time data synchronization

**Payment & Integration**
- Razorpay payment gateway
- Email verification system
- Receipt generation
- SEO optimization
